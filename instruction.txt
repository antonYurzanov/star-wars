1. распаковать архив
2. установить node.js
3. открыть терминал или ide папку с проектом
4. выполнить команду npm install
5. после установки всех пакетов запустить проект командой npm run serve
6. при запуске в терминале указывается ссылка по которой можно открыть проект, типа http://localhost:8080

В проекте реализован функционал создания объекта и отправки его на сервер, путем нажатия на кнопки.
